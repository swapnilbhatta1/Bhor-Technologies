# Spearkentech

Production-ready Next.js landing page for Spearkentech with a neptune-inspired dark theme, modular App Router structure, Tailwind CSS styling, and conversion-focused page sections.

## Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS

## Getting Started

1. Install Node.js 20.9 or newer.
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Notes

- Update the contact details, social links, and metadata domain if needed before deployment.
- Replace the placeholder testimonials and case study details with approved client content for production publishing.
